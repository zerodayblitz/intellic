# Intellic

A dual-purpose privacy and OSINT platform. Anonymize yourself. Investigate anyone.

## What is Intellic?

Intellic is a full-stack web platform built around a single duality: the right to protect your own digital identity, and the tools to investigate anyone else's. It combines a comprehensive privacy hardening suite with a powerful OSINT engine into one unified platform.

## Features

### Vanish — Privacy & Anonymity
- Digital Footprint Scanner
- Data Broker Opt-Out Automation
- OPSEC Score & Privacy Report Card
- Metadata Stripping Tool
- Breach Exposure Tracker
- Real-Time Dox Alert System
- Browser Fingerprint Analyzer
- Threat Model Selector
- And more...

### Uncover — OSINT Engine
- Universal Search Bar
- Username Graph Mapper
- Email Intelligence Module
- Phone Number Deep Dive
- Reverse Image & Face OSINT
- Dark Web Mention Search
- Network Relationship Map
- AI-Powered Identity Reconstruction
- And more...

### Platform-Wide
- The Duality Report — run OSINT on yourself, pipe results into Ghost Mode
- Anonymous account system — no email, no phone number required
- API access for researchers
- Glossary of OPSEC, OSINT, and cybersecurity terms

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express
- **Database:** SQLite (development), PostgreSQL (production)
- **Auth:** Username/password with Argon2id hashing + TOTP 2FA
